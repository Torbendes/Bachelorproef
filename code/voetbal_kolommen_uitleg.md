# Uitleg kolommen voetbal dataset

Dit document bevat een overzicht van **alle kolommen** uit de dataset en
hun betekenis.

| Kolom | Betekenis |
|------+|-----------|
| Div | Divisie / competitie |
| Date | Wedstrijddatum |
| Time | Wedstrijdtijd |
| HomeTeam | Thuisploeg |
| AwayTeam | Uitploeg |
| FTHG | Aantal doelpunten thuisploeg (eindstand) |
| FTAG | Aantal doelpunten uitploeg (eindstand) |
| FTR | Eindresultaat (H = thuiswinst, D = gelijkspel, A = uitwinst) |
| HTHG | Doelpunten thuisploeg bij rust |
| HTAG | Doelpunten uitploeg bij rust |
| HTR | Rustresultaat |
| HS | Aantal schoten thuisploeg |
| AS | Aantal schoten uitploeg |
| HST | Schoten op doel thuisploeg |
| AST | Schoten op doel uitploeg |
| HF | Aantal overtredingen thuisploeg |
| AF | Aantal overtredingen uitploeg |
| HC | Corners thuisploeg |
| AC | Corners uitploeg |
| HY | Gele kaarten thuisploeg |
| AY | Gele kaarten uitploeg |
| HR | Rode kaarten thuisploeg |
| AR | Rode kaarten uitploeg |
