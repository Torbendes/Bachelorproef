\chapter{Feature Engineering}
\label{ch:feature-engineering}

In dit hoofdstuk wordt het proces van feature engineering beschreven, waarbij de ruwe data wordt omgezet in betekenisvolle input voor de machine-learningmodellen.
Feature engineering is een cruciale stap in het machine-learningproces, 
omdat de kwaliteit en relevantie van de features een grote invloed hebben op de prestaties van de modellen.
